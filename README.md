小熊新闻	
=====================


**这是一个使用PhoneGap开发的RSS阅读器，开发于12年6月。**

在此，我非常感谢某生产通信设备的国有企业把这个PhoneGap框架介绍给我，但对其恶意修改比赛规则并且未作出任何道歉，本人深感遗憾。

**现在把整个阅读器开源并且持续维护，鼓励国内开发者使用PhoneGap开发。**
